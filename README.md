* Installation:

    git clone git://github.com/yuchih/dotfiles.git ~/.vim

* Create symlinks:

    ln -s ~/.vim/vimrc ~/.vimrc

* Switch to the `~/.vim` directory, and fetch submodules:

    cd ~/.vim
    git submodule init
    git submodule update
